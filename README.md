# 🌾 Kishan Mitra – Smart Farming System

Kishan Mitra is a modern smart farming web application designed to help farmers access agricultural information and digital farming services through a simple and user-friendly interface. The platform aims to support farmers by providing crop guidance, weather updates, market prices, government schemes, and modern farming techniques in one place.

## 🚀 Features

- 🌱 Smart farming dashboard
- 🌦️ Weather information
- 🌾 Crop recommendations
- 💰 Market price updates
- 🏛️ Government schemes for farmers
- 📚 Farming tips and best practices
- 📱 Fully responsive design
- ⚡ Fast and lightweight UI
- 🎨 Modern React-based interface

## 🛠️ Tech Stack

**Frontend**
- React.js
- Vite
- React Router DOM
- CSS
- Lucide React Icons

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/kishan-mitra.git
```

Navigate to the project folder

```bash
cd kishan-mitra
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Babel](https://babeljs.io/) (or [oxc](https://oxc.rs) when used in [rolldown-vite](https://vite.dev/guide/rolldown)) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/) for Fast Refresh

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).
